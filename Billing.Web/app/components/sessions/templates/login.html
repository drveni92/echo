Our project through sprints

  • Sprint #1

    February - March

    In Spint #1 our focus was on creating database, Web-API on backend: basic CRUD operations, unit tests for entities and some simple mockups.

  • Sprint #2

    March

    Sprint #2 ended up with implementing basic & token authentication, and creating reports on backend.

  • Sprint #3

    March-April

    In Sprint #3 we started with frontend, AngularJS, and created Login with CRUD operations based on API entities.

  • Sprint #4

    April

    Sprint #4 consisted of fixing bugs, creating reports on frontend, improving design and getting ready for our last Sprint.
